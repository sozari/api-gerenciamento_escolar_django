Instalando o Django

Etapas

1. Instalação do venv

        pip install virtualenv

2. Criação de um Ambiente Virtual

        python -m venv .venv

3. Ativação do Ambiente Virtual

        .\.venv\Scripts\activate

4. Atualização do pip

        python.exe -m pip install --upgrade pip

5. Instalação do Django

        pip install django

6. Verificação da Instalação do Django

        django-admin --version

7. Instalação do Conector MySQL

        pip install mysql-connector

8. Criação do Projeto Django

        cd 
        django-admin startproject meuprojeto

9. Ativação do Diretório do Projeto

        cd meuprojeto

10. Exploração da Estrutura do Projeto

O diretório "meuprojeto" contém os seguintes arquivos e pastas:
  • manage.py: Script para gerenciar tarefas do Django, como criar migrações e executar testes.
  • meuprojeto: Diretório principal do seu projeto, contendo o arquivo settings.py com as configurações globais da aplicação.
  • meuprojeto/urls.py: Define as URLs que serão mapeadas para as funções do seu projeto.
  • meuprojeto/wsgi.py: Ponto de entrada para o servidor web do Django.

11. Criação do Aplicativo

        python manage.py startapp meu_projeto

12. Adição do Aplicativo na Configuração do Projeto

No arquivo `settings.py` do projeto "meuprojeto", adicione o aplicativo "meu_projeto" à lista `INSTALLED_APPS`: 
        INSTALLED_APPS = [
            # ... outros aplicativos ...
            'meu_projeto',
        ]

13. Configuração das URLs do Aplicativo

No arquivo `urls.py` do projeto "meuprojeto", importe o aplicativo "meu_projeto" e defina a rota principal: 
        from meu_projeto import views

        urlpatterns = [
            path('', views.index, name='index'),
            # ... outras rotas ...
        ]

14. Criação da Função de Visualização

No arquivo `views.py` do aplicativo "meu_projeto", crie a função de visualização `index` que renderiza o template HTML: 
        from django.shortcuts import render

        def index(request):
            return render(request, 'index.html')

15. Criação do Template HTML (index.html):

Crie o arquivo `index.html` na pasta `templates` do aplicativo "meu_projeto" com o conteúdo HTML desejado para a página inicial.

16. link da folha de estilo:

link rel="stylesheet" href="{{ STATIC_URL }}static/style.css"

17. Execução do Servidor de Desenvolvimento

        cd meuprojeto02
        python manage.py runserver

        Acesse http://127.0.0.1:8000/ em seu navegador para visualizar a página inicial.
